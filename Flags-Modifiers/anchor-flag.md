## Name
anchor flag

## Description
Anchor <i>(flag)</i>

## Syntax (back)
<div>
  A
</div>

## Description (long)
The pattern is forced to become anchored at the start of the search or at the position of the last successful match, equal to a \G.

## Example code (back)
<div>
  /a+/gA
</div>
