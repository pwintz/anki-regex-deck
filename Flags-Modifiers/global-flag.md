## Name
global flag

## Description
for the flag to continue searching after the first match

## Syntax (back)
<div>
  g (flag)
</div>

## Description (long)
<div>
  The "global" flag tells the engine not to stop after the first
  match has been found, but rather to continue until no more
  matches can be found.
</div>

## Example code (back)
/this/g
