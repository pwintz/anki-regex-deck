## Name
unicode newline

## Description
to match a unicode newline

## Syntax (back)
<div><div>\R</div></div>

## Description (long)
Matches any Unicode newline character: <code><CR></code>, <code><LF></code>, <code><CR><LF></code>. Equivalent to <code>[\r\n]</code>.

## Example code (back)
